Student Result Management System

A simple console-based C++ project for managing student results.

Features

- Add multiple student records
- Enter marks for four subjects
- Calculate total marks
- Calculate average marks
- Assign grades automatically
- Display result sheet

Concepts Used

- Variables
- Loops
- Conditional Statements
- Functions
- Strings

Grading System

Average Marks| Grade
90 - 100| A
80 - 89| B
70 - 79| C
60 - 69| D
Below 60| F

How to Run

1. Open the project in Code::Blocks, Dev-C++, Visual Studio, or VS Code.
2. Compile the source code.
3. Run the executable file.
4. Enter student details and marks.
5. View the generated result sheet.
